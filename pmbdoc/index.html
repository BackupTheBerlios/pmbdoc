PMB : Documentation

Ces manuels ont été rédigés avec la grammaire XML DocBook. Les fichiers source XML ont été édités soit dans un éditeur de texte, soit dans XML Mind XML editor (version standard) puis formatés avec l'aide des feuilles de style personnalisées de Norman Walsh sous XFC version standard. Les captures d'écran ont été prises sous Linux avec GIMP, sous Windows avec PrtScr puis GIMP ou sous MacOS X.3, puis éditées et converties avec GIMP.
