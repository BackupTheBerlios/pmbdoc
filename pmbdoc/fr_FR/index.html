Résumé

Ce guide constitue la documentation utilisateur de PMB, système informatisé de gestion de bibliothèques, interface PHP/MySQL libre et opensource. Les utilisateurs auxquels est destiné ce guide sont les membres de l'équipe de la bibliothèque, et non les usagers de celles-ci, les emprunteurs.

Pour plus d'informations, consulter www.sigb.net
